## 📊 СОСТОЯНИЯ ЭЛЕМЕНТОВ

### Кнопка "Избранное"

```
НАЧАЛЬНОЕ СОСТОЯНИЕ:
🤍 (белое сердце) - не в избранном
└─ onClick → ❤️ (красное сердце) - в избранном
```

### Модальное окно

```
ЗАКРЫТО:
display: none
└─ onClick на "Записаться" → ОТКРЫТО
   display: block
   ├─ Полусвежий фон (50% прозрачность)
   ├─ Форма с полями
   └─ onClick на ✕ или вне формы → ЗАКРЫТО
```

### Вкладки в кабинетах

```
ВКЛАДКА НЕАКТИВНА:
display: none
background: белый

└─ onClick на меню → АКТИВНА
   display: block
   background: белый + выделение
```

## 🎯 КЛЮЧЕВЫЕ ТОЧКИ ВЗАИМОДЕЙСТВИЯ

### Главная страница
- **CTA 1:** "Найти тьютора" → Каталог
- **CTA 2:** "Зарегистрироваться" → Регистрация

### Каталог
- **Фильтры:** Изменяют список
- **Сердце:** Добавляет/удаляет из избранного
- **Профиль:** Открывает детальную страницу

### Профиль
- **Избранное:** Добавляет в список
- **Записаться:** Открывает модальное окно
- **Отправить:** Сохраняет бронирование

### Избранные
- **Удалить:** Удаляет из списка
- **Профиль:** Открывает профиль
- **Таблица:** Сравнивает до 3 тьюторов

### Кабинет
- **Вкладки:** Переключают содержимое
- **Действия:** Подтверждение/отмена
- **Сохранение:** Обновляет профиль
